1) To run : 

Click on index.html
To goto student home page : Click on student login and submit button 
To goto faculty home page : Click on faculty login and submit button

(The login validation for both student and faculty has been removed in the source code, so no need of login credentials)

2) Files and Folders

index.html - home page of E-learning portal.
student.html - student home page.
faculty.html - faculty home page.
js - folder which contains jquery files used.
css - folder which contains all css files used.
fonts - folder which contains all custom fonts used.
img - folder which contains all images used.


3) Libraries used

Jquery - for animations.
Bootstrap - for styling the website.
